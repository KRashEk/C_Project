It will produce .S (Assembly) file, if -S option is provided, It will produce .o (Object) file (and delete .S file) if -c option is provided , It will produce executable file (and delete .S and .o files) if neither -S nor -c option is provided , If -o switch is provided to specify custom output file name then executable file will have custom name otherwise default ‘a.out’ name should be used.
